Screens
    - SplashScreen

    - RegScreen
        - modes: Standard, Invalid
        - values: basics, AdIn_Route-Finding

    - WaitingForAuthScreen
        - Loading Screen, circle with lock inside

    - LoginScreen
        - modes: Standard, Invalid

    - HomeScreen
        - modes: Standard, WithData, Offline
        - RecyclerView with surrounding stations and first incoming lines
        - Top Nav Bar with Name, Settings
        - Bottom Nav Bar with WebView TK, Home, Alarm, Pic&Push

    - WebView TK
        - WebView with TK Cur&EP

    - GeneralSettingsScreens
        - values: Type, Study, Group, preferred Lines

    - UnauthSafeScreen

    - PicAndPushScreen
        - [no online data]
        - momentary reminders
        - previous reminders (re-selectable)
    
    - NewPushScreen
        - new / re-use
        - title
        - when
        - alarm?
        [ - location? [allow pictures] ] LtoI
        - back to parent

    - AlarmScreen
        - on/off
        - wakeup x minutes before first connection
        - once / number of alarms
        - sound
        - set manually
            - days?
            - time?
            - feedback!

__INIT__
    - show SplashScreen
    - ask for permissions (once / recurring)
    - check for locally saved config
        - exists?
            - decrypt
            - load data
            - LOGIN_AUTO

        - offline & login timestamp expired?
            - show UnauthSafeScreen

        - no / one-time-login
            - create new empty config
            - REGISTER


REGISTER
    - show GeneralSettingsScreens
    - show RegScreen(Standard)
    - ENTER submit --> check / parse values and call /register
    - show WaitingForAuthScreen
    - AWAIT response --> parse / validate response content
        - ok?
            - HOME

        - no
            - REGISTER(Invalid)


LOGIN_AUTO
    - call /login with user data
    - show WaitingForAuthScreen
    - AWAIT response --> parse / validate response content
        - ok?
            - HOME

        - no
            - clear config>AuthData
            - LOGIN


LOGIN
    - show LoginScreen(Standard)
    - ENTER submit --> check / parse values and call /login
    - show WaitingForAuthScreen
    - AWAIT response --> parse / validate response content
        - ok?
            - HOME

        - no
            - LOGIN(Invalid)


HOME
    - call /envstationdata with GPS location every 10 seconds
    - show HomeScreen(Standard)
    - AWAIT response --> parse / validate response content
        - ok?
            - show HomeScreen(WithData)

        - no
            - RETRY (max. 3 times)
            - show HomeScreen(Offline)

    - ENTER "Pic&Push" --> PIC&PUSH
    - ENTER "AlarmSettings" --> ALARM


[offline-enabled]
PIC&PUSH
    - show PicAndPushScreen
    - ENTER "NewPush" --> show NewPushScreen


[offline-enabled]
ALARM
    - show AlarmScreen
    - ENTER "NewAlarm" --> save values / CREATE new alarm
